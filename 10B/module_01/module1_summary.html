Project Focused Summary

1. Output and Basic Program Structure

A C++ program is made of statements, which are instructions the computer follows step by step. A basic C++ program usually looks like this:

#include <iostream>
using namespace std;
int main() {
    // your statements go here
}

Important parts:

  • #include <iostream> – This lets your program use cout and cin.
  • using namespace std; – This lets you write cout instead of std::cout.
  • int main() { } – This is where the program starts running.

To print something on the screen, use cout:

cout << "Hello world!";

To move to the next line, use endl:

cout << "Hello world!" << endl;
cout << "Welcome!" << endl;

This produces the output:

Hello world!
Welcome!

Use quotation marks for text:

cout << "paycheckAmount";

This prints the word paycheckAmount.

Do not use quotation marks when printing a variable:

cout << paycheckAmount;

This prints the value stored in the variable.

If you need quotation marks inside a string, use \":

cout << "Bob said, \"Wear a jacket!\"" << endl;

This prints:

Bob said, "Wear a jacket!"

For most projects, remember this rule:

cout << "Prompt or message" << endl;

Use one cout statement per line of output when possible. It makes your code easier to read.

2. Variables, Input, Expressions, and Data Types

Variables store values while the program runs.

Common variable types include:

int age;          // whole numbers
double price;     // decimal numbers
char grade;       // one character
string name;      // text

If you use string, include this at the top:

#include <string>

Example:

#include <iostream>
#include <string>
using namespace std;
int main() {
    string name;
    cout << "Enter your name: ";
    cin >> name;
    cout << "Hello, " << name << endl;
}

Input with cin

Use cin to get input from the user:

cin >> hoursWorked;

Usually, put a prompt before cin:

cout << "Enter hours worked: ";
cin >> hoursWorked;

You can read multiple values in one line:

cin >> hours >> payRate;

If the user types 23 10, then hours gets 23, and payRate gets 10.

Assignment Statements

Assignment means “gets”:

paycheck = hoursWorked * payRate;

Read this as: paycheck gets hoursWorked times payRate. Do not think of = as “equals” like in algebra; in C++, = means assign a value.

Arithmetic Operators

OperationC++
Addition+
Subtraction-
Multiplication*
Division/

Example:

total = price * quantity;

Precedence

C++ follows normal math order:

  1. Parentheses first
  2. *, /, % from left to right
  3. +, - from left to right

For example:

result = 12 + 9 / 3;

This gives 15 because division happens before addition.

Integer Division

When dividing two integers, C++ discards the decimal part. For example:

int answer;
answer = 23 / 4; // answer is 5

To get the remainder:

remainder = 23 % 4; // remainder is 3

double

Use double when you need decimals:

double gallons;
double quarts;
quarts = 7;
gallons = quarts / 4; // gallons is 1.75

Be careful mixing int and double:

int i = 11;
int j = 4;
double x;
x = i / j; // x is 2.0, not 2.75

To force decimal division:

x = static_cast<double>(i) / j;

char

Use char for one character:

char grade;
grade = 'A';

Use single quotes for characters and double quotes for strings:

'A', 'x', '?'  
"Hello", "CS 10B"

string

A string stores text:

string message;
message = "hello there";

You can combine strings:

string first = "hello ";
string second = "there";
string full = first + second; // full is "hello there"

Useful string functions:

  • str.length() – Gets the number of characters.
  • str.substr(3, 7) – Gets 7 characters starting at index 3.

Remember: string positions start at 0, not 1.

3. If Statements and Logical Expressions

Use if when your program needs to make a decision.

Basic form:

if (condition) {
    statements;
}

Example:

if (hours > 30) {
    paycheck = paycheck + 100;
}

This means: If hours is greater than 30, add 100 to paycheck.

If-Else

Use if-else when exactly one of two choices should happen:

if (hours <= 40) {
    paycheck = hours * payRate;
} else {
    paycheck = 40 * payRate + (hours - 40) * payRate * 1.5;
}

If the condition is true, the if part runs. If the condition is false, the else part runs.

Else-If

Use else if when there are several possible choices:

if (score >= 90) {
    cout << "A" << endl;
} else if (score >= 80) {
    cout << "B" << endl;
} else if (score >= 70) {
    cout << "C" << endl;
} else if (score >= 60) {
    cout << "D" << endl;
} else {
    cout << "F" << endl;
}

Only one branch runs.

Relational Operators

MeaningC++

Equal to==
Not equal to!=

Important:

  • = means assignment.
  • == means comparison.

Example:

hours = 30; // Put 30 into hours
hours == 30 // Is hours equal to 30?

Logical Operators

MeaningC++
And&&
Or||
Not!

Example: number is between 5 and 15:

(number > 5) && (number < 15)

Example: number is not between 5 and 15:

(number < 5) || (number > 15)

You need a full condition on both sides of && or ||.

Logical Precedence

C++ evaluates logical operators in this order:

  1. !
  2. &&
  3. ||

Use parentheses when you are unsure:

if ((age >= 18) && (hasID == 'Y')) {
    cout << "Allowed" << endl;
}

Project Checklist

For your project, you will probably need to know how to:

  1. Write the basic program structure: 
    #include <iostream>
#include <string>
using namespace std;
int main() {
    
}
  2. Declare variables at the beginning of main(): 
    int hours;
double payRate;
string name;
char choice;
  3. Ask the user for input: 
    cout << "Enter hours: ";
cin >> hours;
  4. Do calculations: 
    paycheck = hours * payRate;
  5. Use if, else if, and else to make decisions: 
    if (score >= 90) {
    grade = 'A';
} else if (score >= 80) {
    grade = 'B';
} else {
    grade = 'F';
}
  6. Print final results clearly: 
    cout << "Your paycheck is $" << paycheck << endl;
  7. Use good style:
    • Use clear variable names.
    • Indent inside braces.
    • Put comments at the top.
    • Use blank lines to separate parts.
    • Use named constants when a number has a clear meaning.

    Example of a named constant:

    const int BONUS_AMOUNT = 100;

    Instead of writing 100 everywhere, use a constant.
